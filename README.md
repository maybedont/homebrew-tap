  ## Install

  ```bash
  brew install maybedont/tap/maybe-dont
  ```

  That's it. Your AI agents now have a chaperone.

  ## Upgrade

  ```bash
  brew upgrade maybe-dont
  ```

  ## Uninstall

  ```bash
  brew uninstall maybe-dont
  ```

  ## What is Maybe Don't?

  Maybe Don't sits between AI agents and the tools they use. Every tool call gets validated against user defined policy.

  Learn more at [maybedont.ai](https://maybedont.ai) or grab the binary directly from [releases](https://github.com/maybedont/releases/releases/latest).
